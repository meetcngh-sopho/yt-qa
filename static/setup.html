This guide shows you how to run a local AI model (Ollama) and expose it to the internet (ngrok) so you can use it with this app from anywhere.

Part 1 — Install Ollama (Local LLM)

Step 1
Download and install Ollama

Go to ollama.com and download for your OS.

Windows Mac Linux
Step 2
Pull the models

Open terminal and run:

ollama pull llama3.2:3b
ollama pull nomic-embed-text

llama3.2:3b is for chat. nomic-embed-text is for embeddings (search).

Step 3
Verify Ollama is running
curl http://localhost:11434/api/tags

You should see a JSON response with your installed models.

Step 4
Use in the app (local only)

In the app settings panel, set:

API URL: http://localhost:11434/v1
API Key: ollama
Model: llama3.2:3b

Part 2 — Expose with ngrok (Access from Internet)

By default Ollama only works on your local machine. ngrok creates a public URL that tunnels to your local Ollama — so you can use it from this app even when deployed on Hugging Face.

Step 1
Install ngrok

Go to ngrok.com → sign up free → download.

Or install via terminal:

# Mac
brew install ngrok

# Windows
winget install ngrok
Step 2
Authenticate ngrok

After signing up, get your auth token from dashboard.ngrok.com and run:

ngrok config add-authtoken YOUR_AUTH_TOKEN
Step 3
Start Ollama + expose with ngrok

Open two terminals:

Terminal 1 — make sure Ollama is running:

ollama serve

Terminal 2 — expose port 11434:

ngrok http 11434
Step 4
Copy your ngrok URL

ngrok will show something like:

Forwarding  https://abc123.ngrok-free.app -> http://localhost:11434

Copy that https://abc123.ngrok-free.app URL.

Step 5
Use ngrok URL in the app settings

In the app settings panel, set:

API URL: https://abc123.ngrok-free.app/v1
API Key: ollama
Model: llama3.2:3b
Note: ngrok free tier URL changes every time you restart ngrok. Update the app settings when it changes.

Part 3 — Use Groq (Free Cloud API, No Local Setup)

If you don't want to run Ollama locally, use Groq — a free cloud API with fast LLMs.

Step 1
Get a free Groq API key

Go to console.groq.com → sign up → API Keys → Create API Key.

Step 2
Use Groq in the app settings
API URL: https://api.groq.com/openai/v1
API Key: your-groq-api-key
Model: llama-3.3-70b-versatile
Free tier: 100,000 tokens/day — enough for ~166 questions/day.
